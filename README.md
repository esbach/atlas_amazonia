# Amazon Atlas Website

A simple, single-page Vue.js website for the Amazon Atlas project.

## Features

- Modern, responsive design
- Single-page application with navigation
- Three main sections: Home, About, and Contact
- Contact form with validation
- Clean, nature-inspired color scheme
- Built with Vue 3 and Vite

## Getting Started

### Installation

First, install the dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

This will start a local development server (usually at `http://localhost:5173`).

### Build

To build for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

To preview the production build:

```bash
npm run preview
```

## Project Structure

- `index.html` - Main HTML file
- `src/` - Source files
  - `main.js` - Application entry point
  - `App.vue` - Main Vue component
  - `style.css` - Global styles
- `vite.config.js` - Vite configuration
- `package.json` - Dependencies and scripts

## Customization

You can easily customize the content by editing the data in `src/App.vue`:

- Site title and branding
- Hero section content
- About section text and features
- Form behavior

## Browser Support

Works in all modern browsers that support ES6+ and Vue 3.
